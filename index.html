GAMES

"A game is a system in which players engage in an artificial conflict, defined by rules, that results in a quantifiable outcome." ( Katie Salen and Eric Zimmerman) "A game is a form of art in which participants, termed players, make decisions in order to manage resources through game tokens in the pursuit of a goal."

Essentially, anything digital, interactive, and used for amusement gets called a game. ... I define this thing -- a game -- as "a system of rules in which agents compete by making ambiguous decisions." Note that "agents" don't necessarily have to both be human, one is often the system (as in a single-player game
